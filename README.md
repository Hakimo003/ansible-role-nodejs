# Ansible Role: NodeJS

An Ansible Role that installs NodeJS for tar/gz on RedHat/CentOS Linux servers.

## Requirements

https://nodejs.org/en/download/
https://nodejs.org/en/download/current/

Select Linux Binaries (x64)

## Dependencies

None

## Example Playbook

 - hosts: all
   roles:
     - role: jenkins-habilitation
